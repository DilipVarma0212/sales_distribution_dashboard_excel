# Sales Distribution Dashboard

*This project focuses on analyzing sales distribution across multiple dimensions using Excel.*


## Table of Contents


### Overview :  

The dashboard provides a comprehensive view of sales performance by comparing target vs actual sales, tracking trends, and identifying top-performing customers, products, and regions.
 

### Business Problem

The business lacks clear visibility into how sales are distributed across time, customers, products, and regions, making it difficult to identify top-performing areas and optimize strategies. This leads to challenges in tracking performance against targets and making data-driven decisions to improve overall sales efficiency.



### Data Source

The primary dataset used for this analysis is the "sales-distribution.xlsx" file, containing detailed information about each employee in the company.
